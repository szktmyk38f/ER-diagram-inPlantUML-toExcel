# ER-diagram-inPlantUML-toExcel

Export ER diagrams written in PlantUML to Excel as data definitions.

# Getting Started

It now refers to test.pu in the same directory.

```bash

Check file structure.

$ ls -l
total 2
-rw-r--r-- excel.py
-rw-r--r-- test.pu
```

Execute Script.

```bash
python excel.py
```

Open `test.xlsx`.
